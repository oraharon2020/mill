# 🏠 המיליון הראשון — הוראות הקמה

מסך יעד חיסכון לטלוויזיה: עיגול התקדמות במרכז, תחנות בדרך למיליון,
קצב חודשי ותחזית, גרף התקדמות, משפטי מוטיבציה ושעון.
מעדכנים מהטלפון — הטלוויזיה מתעדכנת לבד (תוך ~5 שניות).

---

## שלב 0 — לראות שזה עובד (עכשיו, בלי כלום)

```bash
cd ~/Desktop/1MILL
python3 -m http.server 8000
```

ואז בדפדפן:
- **תצוגת טלוויזיה:** http://localhost:8000/?mode=tv
- **תצוגת עדכון (טלפון):** http://localhost:8000/?mode=edit
- להוסיף `&demo=1` בשביל נתוני דוגמה

במצב מקומי הנתונים נשמרים בדפדפן בלבד. לסנכרון טלפון↔טלוויזיה — שלב 1.

---

## שלב 1 — העלאה ל-Vercel (~5 דקות)

לפני ההעלאה: פתח את **config.js** והחלף את `dataId` למחרוזת אקראית
משלך (היא המפתח הסודי לנתונים שלך):

```bash
openssl rand -hex 16   # העתק את התוצאה ל-dataId
```

ואז:

```bash
cd ~/Desktop/1MILL
npx vercel --prod      # (או vercel אם ה-CLI כבר מותקן ומחובר)
```

תקבל כתובת קבועה: `https://<שם-הפרויקט>.vercel.app`

## שלב 2 — חיבור אחסון (לחיצה אחת בדשבורד)

1. https://vercel.com/dashboard → הפרויקט שנוצר → לשונית **Storage**
2. **Create Database → Upstash for Redis** (Free) → **Connect Project**
3. פריסה מחדש כדי שמשתני הסביבה ייקלטו:
   ```bash
   npx vercel --prod
   ```

זהו. אין חוקים, אין מפתחות להעתיק — הפונקציה `api/data.js` מוצאת
את החיבור לבד.

**בדיקה:** פתח את האתר בטלפון ובמחשב במקביל, הוסף הפקדה בטלפון —
במחשב זה יתעדכן תוך כמה שניות (בתחתית המסך יופיע "מסונכרן ☁";
אם כתוב "מצב מקומי" — האחסון עוד לא מחובר).

- **בטלפון:** פתח את הכתובת ושמור כקיצור במסך הבית
  (שיתוף → הוסף למסך הבית). תקבל אוטומטית את מסך העדכון.
- **בטלוויזיה:** `https://<שם-הפרויקט>.vercel.app/?mode=tv`

אחרי כל שינוי בקבצים: `npx vercel --prod` שוב.

## שלב 3 — איזו טלוויזיה לקנות ואיך להגדיר

### המלצת קנייה
- גודל: **32–43 אינץ'** (לחדר — 43 זה מרשים על הקיר).
- פאנל: **LED/LCD רגיל — לא OLED!** המסך הזה סטטי רוב הזמן,
  ו-OLED עלול להיצרב (burn-in). ב-LED אין בעיה.
- מערכת: דגם עם **Google TV / Android TV מובנה** — כדי שאפשר יהיה
  להתקין דפדפן קיוסק. מותגים מתאימים וזולים: Xiaomi, TCL, Hisense
  (סדר גודל 800–1,500 ₪ ל-32–43").
- תלייה: זרוע/תושבת קיר סטנדרטית (VESA).

### הגדרה על הטלוויזיה (Google TV / Android TV)
1. התקן מחנות האפליקציות את **Fully Kiosk Browser** (או TV Bro כחלופה).
2. הגדר בו את כתובת הבית: `https://<שם-הפרויקט>.vercel.app/?mode=tv`
3. ב-Fully Kiosk: הפעל **Launch on Boot** + **Keep Screen On** —
   הטלוויזיה נדלקת ישר על המסך שלך.
4. בהגדרות הטלוויזיה: בטל כיבוי אוטומטי / שומר מסך
   (Screen saver → Off, Sleep timer → Never).

### חלופה סופר-אמינה
כל טלוויזיה (גם "טיפשה") + **Raspberry Pi** (~250 ₪) שמריץ Chromium
במצב kiosk. יתרון: לא תלוי בדפדפן של הטלוויזיה, רץ שנים בלי בעיה.

## שימוש יומיומי

- **עדכון:** פותחים את הקיצור בטלפון → "הפקדה חדשה" → סכום → שמירה.
  הטלוויזיה מתעדכנת תוך שניות, וכשעוברים תחנה (100K, 250K...) — קונפטי 🎉
- **תחנות:** 100K → 250K → 500K → 750K → ★1M, ואחרי המיליון:
  רמה 2 (2M), רמה 3 (3M) וכן הלאה.
- **מצב אורחים:** יש מישהו בסלון ולא בא לך שיראו את הסכומים?
  הכפתור העגול בפינה למעלה בתצוגת הטלפון (🙈) מסתיר את המסך
  בטלוויזיה ומחליף אותו לשעון אווירתי. לחיצה נוספת מחזירה מיד.
  אם שוכחים לבטל — זה חוזר לבד אחרי 3 שעות.
- **שינוי הגדרות:** הכול ב-`config.js` (שם היעד, תחנות, גודל רמה).
  משפטי מוטיבציה: `quotes.js`.
- מי שמעדיף Firebase במקום Vercel — אפשר למלא `firebase:` ב-config.js
  והאפליקציה תעבוד מולו (סנכרון בזמן אמת אמיתי במקום פולינג).
